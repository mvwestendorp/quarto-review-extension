# User Documentation

## Quick Start

1. Click any text to edit
2. Changes are automatically tracked
3. Use `Cmd/Ctrl+F` to search
4. View Change Summary for statistics

## Core Features

**In-Browser Editing** - Click any element to edit with tracked changes

**Change Summary** - View statistics, export summaries from toolbar

**Search** - `Cmd/Ctrl+F` for browser-like search with regex support

**Comments** - Annotate sections with feedback via sidebar

**Undo/Redo** - `Cmd/Ctrl+Z` to undo, `Cmd/Ctrl+Shift+Z` to redo
